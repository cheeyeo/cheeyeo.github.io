### cheeyeo.dev website

Public website for https://www.cheeyeo.dev

* Setup ruby:
```
rvm use ruby-3.0.0 && \
rvm gemset use jekyll2
```

* Running on localhost:
```
bundle exec jekyll serve --baseurl=
```

* Generate using jekyll:
```
bundle exec jekyll build --config _config.yml
```
